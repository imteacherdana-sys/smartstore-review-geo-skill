## ✨ 이 스킬로 할 수 있는 것

엑셀 파일 **하나**만 Claude에 첨부하면 → 끝!

| 단계 | 작업 내용 |
|------|-----------|
| ① 분류 | 별점별 우선순위 자동 분류 (🔴긴급 / 🟡일반 / 🟢감사) |
| ② 답글 작성 | GEO 최적화 답글 전체 자동 생성 |
| ③ 엑셀 저장 | 우선순위 + AI 답글 열 추가 후 저장 |
| ④ HTML 도우미 | 클릭 한 번으로 답글 복사하는 도우미 파일 자동 생성 |

> **GEO(Generative Engine Optimization)란?**  
> AI 검색 엔진이 콘텐츠를 더 잘 이해하고 추천하도록 최적화하는 기법.  
> 리뷰 답글에 상품명·기능·사용 팁을 자연스럽게 녹여, 네이버 AI 쇼핑 노출까지 챙깁니다.

## 🚀 사용 방법 (2가지)

### 방법 1️⃣ — SKILL.md 파일 직접 업로드 (1회성 작업에 추천)

```
1. 아래 [SKILL.md 다운로드] 버튼 클릭
2. Claude.ai 접속 → 새 대화 시작
3. SKILL.md 파일 + 스마트스토어 리뷰 엑셀 파일을 함께 첨부
4. "리뷰 답글 달아줘" 입력 → 자동 실행!
```

### 방법 2️⃣ — Claude 프로젝트에 등록 (반복 사용에 추천 ⭐)

```
1. Claude.ai → 왼쪽 상단 [프로젝트] → [새 프로젝트] 생성
2. 프로젝트 설정 → [지식 추가] → SKILL.md 업로드
3. 이후 이 프로젝트에서 엑셀만 첨부하면 매번 자동 실행!
```

> 💡 프로젝트 방식을 사용하면 SKILL.md를 매번 올리지 않아도 돼서 훨씬 편해요!

## 📥 스마트스토어 리뷰 엑셀 다운로드 방법

1. [판매자센터](https://sell.smartstore.naver.com) 접속
2. 왼쪽 메뉴 → 문의/리뷰관리 → **리뷰 관리**
3. 필터 설정:
   - 리뷰작성일: **1년** 선택
   - 답글여부: **미등록** 선택
4. 초록색 **검색** 클릭
5. 목록 하단 **엑셀다운로드** 클릭

## ⚠️ 사용 전 확인 사항

- Claude.ai **Pro 플랜** 이상 권장 (파일 업로드 + 긴 대화 처리)
