## [사전 준비] 엑셀 파일 다운로드 방법
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
아래 방법으로 엑셀 파일을 먼저 준비해주세요:

1. 판매자센터 접속: sell.smartstore.naver.com
2. 왼쪽 메뉴 → 문의/리뷰관리 → **리뷰 관리**
3. 상단 필터에서:
   - 리뷰작성일: **1년** 선택 (오래된 리뷰까지 포함)
   - 답글여부: **미등록** 선택
4. 초록색 **검색** 버튼 클릭
5. 목록 오른쪽 하단 **엑셀다운로드** 클릭
6. 다운로드된 파일을 여기에 첨부

엑셀 파일 준비됐으면 첨부해주세요! 📎

## [STEP 1] 파일 확인 및 우선순위 분류
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
파일을 열어서 확인 후 아래 기준으로 분류:

🔴 긴급: 별점 1~2점, 불만·불량·환불 언급
🟡 일반: 별점 3~4점, 아쉬운 점 있지만 대체로 만족
🟢 감사: 별점 5점, 만족 리뷰

분류 결과를 표로 보여주고 사용자 확인 받은 후 진행.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
## [STEP 2] GEO 최적화 답글 작성
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔴긴급 → 🟡일반 → 🟢감사 순서로 전체 답글 작성.

### 작성 원칙
1. 상품명·모델명 자연스럽게 **1회** 포함
2. 핵심 기능·소재 **1~2가지** 구체적 언급
3. 고객 사용 상황을 **구체적으로** 반영
4. 사용 팁 **한 줄** 추가
5. **3~5문장**, 간결하게
6. 이모지 **1~2개**

### 별점별 전략
■ **5점**: 감사 + 리뷰 공감 + 상품 특징 + 사용 팁
■ **4점**: 감사 + 아쉬운 부분 공감 + 상품 정보 + 개선 의지
■ **3점 이하**: 공감·사과 + 원인 설명 + 1:1 문의 유도
  ⚠️ 방어적·변명·고객 탓 **절대 금지**

### 절대 쓰지 말 것
❌ "감사합니다! 다음에도 이용해주세요" (정보 없는 형식적 답변)
❌ 키워드 반복 도배
❌ 리뷰 내용과 무관한 답글
❌ 과도한 이모지 (3개 이상)
❌ 3점 이하에 가벼운 이모지 사용

## [STEP 3] 엑셀 저장
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
원본 엑셀에 두 열 추가 후 저장:
- **"우선순위"**: 🔴긴급 / 🟡일반 / 🟢감사
- **"AI 답글 (GEO)"**: 작성한 답글 내용

파일명: `review_YYYYMMDD_GEO완료.xlsx`

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
## [STEP 4] HTML 답글 복사 도우미 생성
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
엑셀 데이터를 기반으로 HTML 답글 복사 도우미 파일을 자동 생성한다.

### HTML 도우미 기능:
- 🔴긴급 → 🟡일반 → 🟢감사 순서로 자동 정렬
- **[📋 답글 복사]** 버튼 클릭 → 클립보드 자동 복사
- 우선순위별 필터링 (긴급/일반/감사/미완료/완료)
- 진행률 바 (몇 건 완료/남은 건수)
- [등록완료 체크] 버튼으로 진행 추적
- 리뷰 글번호 표시 (판매자센터에서 검색용)

### HTML 생성 방법:
Python 스크립트로 엑셀 데이터를 읽어 JSON으로 변환 후
HTML 파일에 내장하여 단일 파일로 생성한다.

파일명: `답글복사_도우미_YYYYMMDD.html`

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
## [STEP 5] 판매자센터 답글 등록 안내
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
HTML 도우미 파일과 판매자센터를 나란히 띄워서 작업:

### 등록 순서:
1. HTML 도우미 파일 더블클릭으로 열기
2. 판매자센터 → 리뷰 관리 → 답글미등록 필터 → 검색
3. 첫 번째 리뷰의 **리뷰내용** 클릭 → 팝업 열림
4. HTML 도우미에서 **[📋 답글 복사]** 클릭
5. 팝업의 **판매자답글** 칸에 **Ctrl+V** 붙여넣기
6. **[답글 등록]** 클릭
7. **[다음 리뷰 >]** 클릭 → 4번부터 반복
8. HTML 도우미에서 **[등록완료 체크]** 클릭

💡 팁: `Win+←` / `Win+→` 으로 화면 분할하면 빠르게 작업 가능!
